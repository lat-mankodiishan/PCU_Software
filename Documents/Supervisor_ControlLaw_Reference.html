Supervisor & Control Law — current code reference

Role. supervisor_task is the powertrain state machine. It consumes external commands (RC via Lua, MP param.GetSet, GPIO PA7), enforces fault policy, sequences the BLDC↔FOC motor swap on CRANK→RUN, drives the rectifier setpoint, mirrors engine throttle to PWM, and emits a heartbeat for pdb_task. control_law.c holds the V1 hybrid controller — currently wired but disabled via RUN_USE_V1_CONTROLLER = 0.

Task priority: osPriorityHigh (5). Stack 256 words (1 KB). Switch GPIO: PA7 input.

Internal state (static)

  • s_v1_state, s_v1_params — #if RUN_USE_V1_CONTROLLER only.
  • s_prev_engine_state — for transition-edge detection.
  • s_swap_phase, s_swap_tick — sub-state machine for CRANK→RUN.
  • s_switch_prev — debounce for GPIO PA7 rising edge.

Severity policy (bench mode = all WARN)

fault_severity_t: SEV_NONE < SEV_WARN < SEV_DEGRADE < SEV_FORCE_OFF < SEV_FORCE_FAULT. Lookup table s_fault_severity[16] maps each fault bit to a severity. worst_severity(f) walks the 16 bits, returns the highest. Currently every entry is SEV_WARN (bench override). Trailing comment in each row preserves the intended-for-flight severity.

Main task loop @ 5 Hz

  1. Snapshot engine.state, engine.req, fault_bits, expt.active under g_pt_mtx.
  2. Detect faults: detect_faults(now, engine_state) (see below). Re-read fault_bits.
  3. Severity policy: sev = worst_severity(faults).
    • SEV_FORCE_FAULT + not already FAULT → pt_set_engine_state(ENGINE_FAULT), abort swap, clear req.
    • SEV_FORCE_OFF + not OFF/FAULT → pt_set_engine_state(ENGINE_OFF), abort swap, clear req.
  4. External request handler:
    • engine.req != engine.state:
      • req == RUN: only fires swap if currently CRANK and swap IDLE.
      • Else (OFF / CRANK / FAULT etc.): apply direct via pt_set_engine_state, abort swap, clear req.
    • req == engine.state: clear req.
  5. Compute degrade = (sev >= SEV_DEGRADE), switch_now = PA7 read.
  6. Transition entries when engine_state != s_prev_engine_state:
    • Entering RUN: on_run_entry() (FOC inv + V1 init when enabled).
    • Entering CRANK: re-arm GPIO switch (s_switch_prev = switch_now).
    • Leaving CRANK/RUN: cancel any in-progress swap.
  7. Rising-edge detect on PA7 → if (not expt_active, engine_state CRANK, swap IDLE), fire s_swap_phase = SWAP_BLEED.
  8. Dispatch:
    • expt_active: step aside (experiment owns the wire).
    • degrade: safe_idle().
    • swap_step(now) == true: swap drove the wire this tick.
    • Else switch on engine_state: CRANK→crank_hold, RUN→run_fixed_setpoint (or run_v1 when enabled), WARMUP/COOLDOWN→I=0 idle, OFF/FAULT→safe_idle.
  9. Contactor policy: batt_close = !engine_fault, rect_close = !engine_fault && !degrade. pt_set_contactor_cmds(batt, rect) mirrors via pdb_task GPIOs.
  10. Throttle PWM mirror: read engine_throttle.req_pct_x100, call pt_set_engine_throttle_pct_x100(req) which writes TIM1_CH1 CCR.
  11. Heartbeat: g_pt.supervisor_heartbeat++ (consumed by pdb_task hang detector).
  12. Watchdog: watchdog_refresh() kicks IWDG.
  13. osDelayUntil(next).

control_law.c / control_law.h (V1 controller)

Status: source-present, default params populated, not on the wire right now (gated by RUN_USE_V1_CONTROLLER = 0 in supervisor). When flipped on, run_v1() in supervisor calls control_law_v1_step() every 200 ms.

Architecture

Single cascade: I_bat PID → I_rect_demand → P_rect = I × V → LUT(P_rect) → (duty, theta_engine). All Q15 fixed-point, single 16-bit working integers, no float. Output is rectifier duty + engine throttle %.

Step algorithm — control_law_v1_step(st, in, p, out)

  1. I_bat EMA: initialise i_bat_filt_cA from raw on first call. Otherwise filt += alpha_q15 × (meas - filt) >> 15. Clamp to int16.
  2. Effective I_bat_ref: start from i_bat_ref_cA. If last-tick i_rect_demand_cA > i_rect_max - sat_margin, add i_bat_ref_boost_cA so the battery picks up the slack. (Cyl-temp boost migrated to fault path; theta_sat boost not yet wired.)
  3. PID: err = filt - i_bat_ref_eff. p_term = (err × kp_q15) >> 15. i_term += (err × ki_q15) >> 15 clamped to ±i_term_max_cA. d_term = (filt - prev_filt) × kd_q15 >> 15. PID output is the increment: target = i_rect_demand + p + i + d.
  4. Anti-stall RPM clamp: if gen_rpm < omega_floor_rpm → target = 0. Else if < omega_min_rpm → linear derate scaling target by (rpm − floor) / (min − floor).
  5. Clamp + slew: clamp target to [0, i_rect_max_cA]. Slew-limit delta vs last tick to ±i_rect_slew_cA. Store as new i_rect_demand_cA.
  6. P_rect: p_rect_W = max(0, i_rect_demand_cA) × v_bus_cV / 10000.
  7. LUT: control_law_v1_lut(p_rect_W, &duty_x10000, &theta_pct_x100).
  8. Output: fill ctl_v1_output_t.

LUT — control_law_v1_lut() (placeholder)

Linear ramp capped at V1_LUT_P_RECT_MAX_W = 4500 W:

  • duty_x10000 = clamp(p / 4500) × 9500 (caps duty at 95 %)
  • theta_pct_x100 = clamp(p / 4500) × 10000 (caps theta at 100 %)

Intended to be replaced with a bench-calibrated piecewise table from a dyno sweep.

Sign / unit conventions

  • i_bat_meas_cA is +discharge (positive = battery draining). PID error = filtered − ref, so positive error means draining too fast → demand more rectifier current.
  • Outputs are positive-only after the LUT (no regen path), even though i_rect_demand_cA is signed int16 internally.
  • All percent fields use 0.01 %/LSB: 0..10000 = 0..100 %.
  • Currents use 0.01 A/LSB (cA); voltages 0.01 V/LSB (cV).

Init & default helpers

  • control_law_v1_init(st) — zeros integrator + EMA state; init_done = false so the EMA snaps to the first sample.
  • control_law_v1_default_params(p) — populates the table above; meant as starting points, not flight gains.

End-to-end paths

RC switch → engine action:
Lua publishes ArrayCommand actuator_id=8 → fc_link_task → pt_request_engine_state() → supervisor's external request handler → OFF/CRANK direct, or RUN via SWAP_BLEED → LOCK → PRIME → ENGINE_RUN.

Engine state → wire (per tick):
engine.state → supervisor dispatch → safe_idle / crank_hold / swap_step / run_fixed_setpoint → pt_set_setpoint* → g_pt.rect_cmd → rectifier_task (200 Hz) → VESC over CAN2. Engine throttle: g_pt.engine_throttle.req_pct_x100 → pt_set_engine_throttle_pct_x100 → TIM1_CH1 PWM.

Fault → action:
Per-task or detect_faults set a bit in g_pt.fault_bits → supervisor reads → worst_severity picks the strongest → SEV_DEGRADE safe_idles + opens rect contactor; SEV_FORCE_OFF sets ENGINE_OFF; SEV_FORCE_FAULT sets ENGINE_FAULT + opens battery contactor. Currently every fault is demoted to SEV_WARN (bench mode), so detection runs but no action is taken.
